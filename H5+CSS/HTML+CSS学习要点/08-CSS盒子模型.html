页面布局三大核心：盒子模型、浮动、定位

网页布局过程：利用CSS摆盒子。
1.准备相关网页元素，基本都是盒子Box；
2.利用CSS设置好盒子样式，并摆放到相应位置；
3.往盒子里面装内容。

1. 盒子模型

1.1 盒子模型组成：CSS盒子模型本质是一个盒子封装周围的HTML元素，包括：边框、外边距、内边距、实际内容。

1.2 边框:border

  • border-width:边框粗细，一般单位是px
  • border-style:边框样式
  • border-color:边框颜色，不设置默认为黑色

边框复合性写法：没有顺序，通俗性约定粗细、样式、颜色

边框可以分开单独写：border-top、bottom、right、left

边框层叠

1.3 表格细线边框：border-collapse="collapse"为合并边框

边框会影响盒子实际大小： 1.测量盒子大小时，不量边框； 2.测量包括边框，则在width/height中减去边框宽度。

1.4 内边距：padding，边框与内容之间的距离，padding-top/bottom/right/left

1.4.1 内边距复合写法

  • 1个值 5px:上下左右各5像素边距
  • 2个值 5px 10px：上下内边距5像素，左右内边距10像素
  • 3个值 5px 10px 20px：上边距5，左右10，下边距20
  • 4个值 5px 10px 20px 30px：上5，右10，下20，左30（顺时针）
  • 内边距会影响盒子实际大小，如果盒子已经有高宽，再指定内边框会撑大盒子
  • 保证盒子高宽不变，则在width/height中减去内边距大小。

不用给盒子宽度，直接给padding值撑开盒子宽度，小方块为两个padding+文字大小

1.4.2 盒子本身没有指定width/height属性，则内边距不会影响盒子大小

盒子内容是content

内容是盒子

1.5 外边距：margin；盒子与盒子间的距离，与padding写法完全一致（上下左右，复合写法）

1.5.1 外边距典型应用:让块级元素盒子水平居中，前提条件：

  • 盒子必须指定了width
  • 盒子左右的外边距都设置为auto（自动）

1.5.2 让行内/行内块元素水平居中：给父元素添加 text-align：center

1.5.3 外边距合并

嵌套块元素垂直外边距塌陷：对于两个嵌套关系（父子关系）的块元素，父元素上有外边距，子元素也有，此时父元素会塌陷较大的外边距值:解决方案如下

  • 为父元素定义上边距
  • 为父元素定义上内边距
  • 为父元素添加 overflow:hidden
  • 如果盒子有浮动、固定、绝对定位的盒子就不会有塌陷问题

1.6 清除内外边距

不同浏览器默认的网页元素一般都带有默认内外边距，因此在布局前要清楚网页元素内外边距

注意:行内元素为了照顾兼容性，尽量只设置左右内外边距，不设置上下内外边距，但是转换为块级元素和行内块元素就可以了

1.7 圆角边框（重点）:border-radius：length;设置元素的外边框圆角,length单位为数值或者百分比形式；数值越大，圆角越大； radius:圆的半径原理：圆与边框交集形成圆角效果

1.7.1 做一个圆形，让半径设置为正方形高度/宽度一半

1.7.2 圆角矩形设置为高度的一半

1.7.3 可以写4个值，按4个顺时针变化；2个为对角数值

1.7.4 分开写法：border-top-left-radius/border-top-right-radius/border-bottom-right-radius/border-bottom-left-radius;top和left值不能颠倒

1.8 盒子阴影（重点）：box-shadow;盒子阴影不占用空间，不会影响其他盒子排列

  • h-shadow:必需，水平阴影位置，允许负值
  • v-shadow:必需；垂直阴影的位置，允许负值
  • blur:可选，模糊距离
  • spread:可选；阴影的尺寸
  • color:可选；阴影颜色
  • inset:可选；将外部阴影（outset）改为内部阴影;默认外部阴影，但是不准写外部阴影单词，否则阴影无效

1.9 文字阴影：text-shadow

  • h-shadow:必须，水平阴影位置，允许负值
  • v-shadow:必需；垂直阴影的位置，允许负值
  • blur:可选，模糊距离
  • color:可选；阴影颜色
