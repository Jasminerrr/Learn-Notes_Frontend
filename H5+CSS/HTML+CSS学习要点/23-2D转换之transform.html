1. 2D转换 transform

转换（transform）是CSS3中具有颠覆性的特征之一,可以实现元素的位移、旋转、缩放等效果.(可以简单理解为变形)

1.1 移动translate：

定义 2D 转换中的移动，沿着 X 和 Y 轴移动元素 1.translate最大的优点：不会影响到其他元素的位置!!!! 2.translate中的百分比单位是相对于自身元素的 translate:(50%,50%); 3.对行内标签没有效果

transform: translate(x,y); 或者分开写: transform: translateX(n); transform: translateY(n);

1.2 旋转 rotate

1. 语法 transform:rotate(度数) 2.rotate里面跟度数， 单位是 deg 比如 rotate(45deg) 3.角度为正时，顺时针，负时，为逆时针 4.默认旋转的中心点是元素的中心点

1.3 2D转换中心点 transform-origin

1.语法：transform-origin:x y; 2.注意后面的参数 x 和 y 用空格隔开 3. x y 默认转换的中心点是元素的中心点 (50% 50%) 4.还可以给x y 设置 像素 或者 方位名词 （top bottom left right center）

1.4 缩放：scale

语法：transform:scale(x,y);

1.注意其中的x和y用逗号分隔 2.transform:scale(1,1) ：宽和高都放大一倍，相对于没有放大 3.transform:scale(2,2) ：宽和高都放大了2倍 4.transform:scale(2) ：只写一个参数，第二个参数则和第一个参数一样，相当于 scale(2,2) 5.sacle缩放最大的优势：可以设置转换中心点缩放，默认以中心点缩放的，而且不影响其他盒子

2. 2D转换综合写法

1.同时使用多个转换，其格式为：transform: translate() rotate() scale() ...等， 2. 其顺序会影转换的效果。（先旋转会改变坐标轴方向） 3. 当我们同时有位移和其他属性的时候，记得要将位移放到最前
