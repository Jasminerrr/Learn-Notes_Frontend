1.3 伪元素选择器（重点）

::before 在元素内部的前面插入内容
::after 在元素内部的后面插入内容
都是盒子，也是父元素的孩子 before 和 after 创建一个元素，但是属于行内元素
新创建的这个元素在文档树中是找不到的，所以我们称为伪元素
语法： element::before
before 和 after 必须有 content 属性，伪元素选择器和标签选择器一样，权重为 1
