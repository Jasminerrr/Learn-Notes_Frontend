PS切图：图层切图、切片切图、PS插件切图等

jepg：色彩信息保留好，高清，色彩丰富，有背景颜色，png:可为透明背景，gif：色彩少动图效果和透明背景

1. 图层切图：右击图层-快速导出为PNG
大多情况需要合并图层再导出：
选中图层：菜单-合并图层（ctrl+e）-右击导出png；

2. 切片切图
1.利用切片工具手动划出
2.导出选中图片：文件菜单-导出-储存为web设备所用格式-选中要的图片格式-储存-切片选择为选中的切片-保存。

3. PS插件切图： Cutterman是一款运行在Photoshop中的插件，能自动将需要的图层进行输出，代替其他两个方式
网址 插件要求PS必须是完整版，不能是绿色版
