1.元素显示模式：了解标签特点更好的布局网页：块元素和行内元素

块元素和行内元素

1.1 块元素：h1~h6,p,div,ul,ol,li等

特点：1.独占一行； 1. 高、宽、外边距、内边距都可控制； 2. 宽度默认是容器（父级宽度）的100%； 3. 是一个容器及盒子，里面可以放行内元素或块级元素。

注意： 文字类的元素不能使用块级元素； p标签主要存放文字，所以不能放块级元素，特别是div; 同理h1~h6等都是文字类标签，也不能放块级元素。

1.2 行内元素：a,strong,b,em,i,span等（行内也可叫内联元素）

行内元素特点： 1. 相邻行内元素在一行，一行可显示多个； 2. 高、宽直接设置无效，不能直接设置； 3. 默认宽度就是本身内容得宽度； 4. 只能容纳文本或其他行内元素。

注意： 链接里面不准放链接； 特殊情况链接a可以放块级元素，但是a转换一下块级模式最安全。

1.3 行内块元素：,,,同时具有块元素和行内元素特点

行内块元素特点： 1. 相邻行内元素（行内块）在一行上，但是之间会有空白缝隙；一行可显示多个（行内元素特点） 2. 默认宽度就是本身内容得宽度（行内元素特点） 3. 高、行高、外边距、内边距都可控制（块级元素特点）；
