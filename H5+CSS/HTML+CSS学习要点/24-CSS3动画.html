动画

可通过设置多个节点来精确控制一个或一组动画，常用来实现复杂的动画效果。 相比较过渡，动画可以实现更多变化，更多控制，连续自动播放等效果。

1.动画的基本使用

1.1 先定义动画 2.2 再使用（调用）动画
动画序列：

1. 0% 是动画的开始，100% 是动画的完成。这样的规则就是动画序列。 2. 在 @keyframes 中规定某项 CSS 样式，就能创建由当前样式逐渐改为新样式的动画效果。 3. 动画是使元素从一种样式逐渐变化为另一种样式的效果。可以改变任意多的样式任意多的次数。 4. 请用百分比来规定变化发生的时间，或用关键词 "from" 和 "to"，等同于 0% 和 100%

  • 可以做多个状态的变化，keyframes(关键帧)
  • 百分比要是整数
  • 里面的百分比是总时间的划分（此案例10s）

2. 动画常用属性

动画名称 animation-name: move; 持续时间 animation-duration: 10s; 运动曲线 （默认加速ease，匀速linear） animation-timing-function: ease; 何时开始 (默认0开始) animation-delay: 1s; 播放次数,iteration重复的、count次数（默认1，无限是infinite） animation-iteration-count: infinite; 是否反方向播放，默认normal,反方向是alternate animation-direction: alternate; 结束后状态，默认回到起始状态 backwards/停留在结束状态 保持forwards animation-fill-mode: forwards; 鼠标经过div，停止动画，鼠标离开就继续动画(默认running，暂停paused) animation-play-state: paused;

3. 动画简写属性

animation：动画名称 持续时间 运动曲线 何时开始 播放次数 是否反方向 动画起始或者结束的状态
1. 前两个必须要写，默认属性可以省略
2. 简写属性里面不包含 animation-play-state
3. 暂停动画：animation-play-state: puased; 经常和鼠标经过等其他配合使用

4. 速度曲线调节

animation-timing-function：规定动画的速度曲线，默认是“ease”
