1. vw / vh 是什么

1.vw/vh是一个相对单位（类似em和rem相对单位） vw:viewport width 视口宽度单位
vh:viewport hight 视口高度单位
相对视口得尺寸计算结果： 1vw = 1/100 视口宽度
1vh = 1/100 视口高度
注意：和百分比布局区别：百分比是相对父元素来说，而vw/vh总是针对于当前视口来说！！！！

2. 开发中使用vw，像素大厨有哪些改动

1. 把模式改为2x模式
2. 确定设计稿视口高度。比如375。
3. 直接使用 测量数值 / (视口宽度/100)
4. 比如：50/（375/100）

3. 注意事项

本质是根据视口宽度来等比缩放页面元素高度和宽度，实际开发使用vw就够了，vh很少使用。 兼容性网站：http://caniuse.com/
