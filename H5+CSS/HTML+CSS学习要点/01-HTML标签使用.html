5. 表单标签

5.1 表单的组成：表单域；表单控件（表单元素）；提示信息

5.1.1 表单域="form"标签；指包含表元素的区域，实现用户信息的收集和传递，将范围内的表单元素信息提交给服务器

5.1.2 表单元素：1.input输入表单元素；2.selcet下拉表单元素；3.textarea文本域元素

1."input /"标签为单标签，用于收集用户信息；用"type"不同的属性值用来指定不同的控件；除type外还有别的属性

type属性如下：
button:按钮，用于通过JavaScript启动脚本
checkbox：复选框,可多选
file：文件域；输入字段和“浏览”按钮，供文件上传
hidden：隐藏字段
image：图像提交按钮
password：密码字段，字符被掩码
radio：单选按钮
reset：重置按钮；会清除表单中所有数据，还原表单的重置状态
submit：提交按钮；把表单数据发送到服务器
text:文本，默认宽度为20个字符
除type外的其他属性常用如下：
name：input元素的名称,自定义；要求单选和复选按钮要有相同的name值
value：规定input元素的值
name和value时每个表单都有的属性值，主要给后台人员使用,文本框才能看到效果
checked：规定此input元素首次加载时应当被选中
maxlength:正整数；规定输入字段中字符的最大长度
