1. 层叠性

相同的选择器给设置相同的样式，此时样式会覆盖（重叠）另一个冲突的样式； 层叠性主要解决样式冲突问题

层叠性原则： 1.样式冲突时，就近原则，执行离结构近的样式； 2.不冲突，不会层叠。

2. 继承性

CSS的继承：子标签会继承父标签的某些样式，如文本颜色和字号（text-,font-,line-元素开头和color属性） 恰当地使用继承可以简化代码，降低CSS的复杂性

2.1 行高的继承
  • 没有指定大小，继承body的文字大小和行高12px/1.5

3. 优先级

选择器相同，则执行层叠性； 选择器不同，则根据选择器权重执行。 继承或者*（通配符） 0000 元素（标签）选择器，伪元素选择器 0001 类选择器，伪类选择器 0010 ID选择器 0100 行内样式 style="" 1000 !important重要的 无穷大

优先级注意点： 继承的权重是0，如果该元素没有直接选中，不管父元素权重多高，子元素得到的权重都是0； 权重叠加：如果是复合选择器，则会有权重叠加，需要计算权重。
